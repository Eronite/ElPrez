. Pourquoi j'ai précisé "FRAM" ?
Beaucoup de cartouches AliExpress "pas chères" utilisent un système appelé Batteryless Save qui patche ta ROM pour sauvegarder directement dans la puce Flash. C'est à éviter pour ton jeu, car ton code fait des écritures fréquentes (fin de mois, construction). La Flash s'use vite, alors que la FRAM ou la SRAM est faite pour ça.

2. Le Mapper MBC5
Ne les laisse pas te proposer du MBC1 ou du MBC3. Ton code compilé avec GBDK est optimisé pour le MBC5. C'est le plus stable et il gère mieux les banques de données que tu utilises pour Penultimo et tes cartes.

3. Le "Golden Sample"
N'accepte jamais une commande de 200 pièces sans avoir testé un échantillon final.

Le test ultime : Lance le jeu, construis une ferme, attends que le mois passe (sauvegarde auto), éteins la console, retire la cartouche, attends 10 minutes, remets-la. Si ta ferme est toujours là, c'est gagné.

4. Le sticker (L'étiquette)
La plupart des fabricants peuvent imprimer l'étiquette pour toi si tu fournis un fichier .psd ou .png en haute résolution (300 DPI). C'est souvent inclus dans le prix ou pour quelques centimes de plus.

C'est la touche finale qui rendra ton jeu professionnel. Une étiquette bien conçue fait toute la différence quand on tient la cartouche en main.

Voici les spécifications techniques pour créer ton visuel.

1. Dimensions de l'étiquette (Standard Game Boy)
Les étiquettes de cartouches Game Boy originales et de reproduction respectent généralement ces dimensions :

Dimensions physiques : 42 mm x 37 mm.

Coins supérieurs : Ils doivent être légèrement arrondis (rayon d'environ 2 mm).

Zone de sécurité : Garde tes textes et logos importants à au moins 2 mm du bord pour éviter qu'ils ne soient coupés lors de l'impression ou de la découpe.

2. Configuration du fichier (Canvas / Photoshop)
Pour une impression nette (sans flou), utilise ces réglages :

Résolution : 300 DPI (Indispensable).

Taille en pixels : 496 x 437 pixels.

Mode colorimétrique : CMJN (CMYK) si ton imprimeur le demande, sinon le RVB (RGB) classique fera l'affaire pour la plupart des fabricants de cartouches flash.

3. Éléments à inclure pour le style "Rétro"
Pour que ton jeu ressemble à une sortie officielle de 1950 (ou de l'époque Game Boy), voici ce que tu peux ajouter :

Le titre : Bien lisible au centre.

L'illustration : Un beau dessin de Penultimo ou une vue aérienne de ta ville.

Le Code Produit (Optionnel) : En bas à droite, tu peux inventer un code du type DMG-TRP-FRA (Tropico Game Boy France).

Le Logo de ton studio : Un petit logo fictif en bas à gauche.

Le "Seal of Quality" : Tu peux créer ton propre sceau "Approved by El Presidente" pour remplacer le logo Nintendo.

4. Astuce pour la découpe
Si tu décides d'imprimer les étiquettes toi-même sur du papier autocollant brillant :

Utilise un cutter de précision (type X-Acto) et une règle métallique.

Pour les coins arrondis, utilise un coupe-coin de petit diamètre ou fais-le très délicatement aux ciseaux.
